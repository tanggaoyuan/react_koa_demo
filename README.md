服务端项目来自 [https://github.com/SimpleCodeCX/koa-ts-base-template](https://github.com/SimpleCodeCX/koa-ts-base-template)


```node
//启动服务器开发
npm run server:dev
//打包服务器代码 在打包后的项目用npm start启动
npm run server:build
// 启动前端开发服务
npm run web:dev
// 打包前端代码
npm run web:build

// 前端和koa同服开放
npm run dev
// 前端和koa服务打包
npm run build
```
